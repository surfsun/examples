# 2022年
2022年11月30日，OpenAI公布了一个通过由GPT-3.5系列大型语音模型微调而成的全新对话式AI模型ChatGPT，它不仅能进行自然的多轮对话、高效的精准问答，还能生成编程代码、电子邮件、论文、小说等各类文本。这是一切风暴的起点。

# 2023年

3月24日：开始使用gpt的网页版，用来尝试生成代码，能生成，但是还不能用于公司的项目代码编写。

4月26日：10美元一个月的编程ai助手copilot，我试了一下。 ai开发尝鲜，甚至在某次沟通中提到这个工具，获得认可。

    23年6月份的时候ChatGPT发布了一个协议模型调用工具的协议叫Function Call

8月10日：idea中安装bito插件，尝试使用ai写代码，效果都还不太行

这个阶段，网页和插件中，都是提问回答的方式，刚开始还不能引用文件，要复制内容。 后面可以开始引用文件，文件夹，图片，生成的代码还不太满足开发期望。


# 2024年
    24年AI编程随着Claude3.5 sonnet的出现迈入了一个新的台阶，Cursor代表的AI编程工具开始走进大众的视野
    24年11月份，Claude团队带来了第二个基准：MCP（Model Context Protocol）, MCP的本质是为了让模型“看得见世界，动得了手脚”

# 2025年

3月18日，cursor开始流行，新用户注册了有免费额度，然后有bug是额度用完之后可以重新注册，然后继续使用免费额度。
cursor使用的模型是claude，是专门针对代码生成的模型，生成的代码效果比较好。这个阶段开始使用cursor进行小程序页面开发，生成的代码质量不错，能够满足开发需求。

用到5月份，开始封禁bug，逐渐放弃cursor

7月14日：开始尝试Claude Code

        Claude Code 是一个由 Anthropic 开发的基于终端的 AI 编码工具，基于 Claude 4 Sonnet 模型
        2025/02/24，research preview 发布
        2025/05/02，Claude Max 订阅用户可用
        2025/05/22，Claude Code 1.0 版本NPM包发布
        2025/05/23，Claude Code 全面开放 以及 Claude4发布会
        2025/06/04，Claude Pro 订阅用户可用

国内有免费的中转站，提供了免费额度。命令行里安装，然后配置api-key即可使用。后因claude封国内ip，逐渐放弃使用。

10月18：了解到codex,闲鱼上可以加入其他人的工作空间，在idea中有codex的插件，配合git的diff，用起来体验还不错。 claude和codex在idea中用插件的
方式使用，避免同时开idea和cursor的问题。

11月28日：尝试使用google的antigravity，这个工具的使用体验不错，生成的代码质量也比较高，能够满足开发需求。
antigravity跟进了skill功能，尝试后感觉和之前的开发已经很不一样了。skill可以把前人的经验凝结下来，后续边用边优化，生成的代码质量越来越高了。
